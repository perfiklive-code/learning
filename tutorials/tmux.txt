sudo apt install tmux        установка
ctrl + b  ключова комбінація клавіш після якої вводяться всі команди
:kill-session     закрити сесію і вийти
"  відкрити горизонтальну вкладку 
%  відкрити вертикальну вкладку
стрілочка переключення між вікнами
:resize-pane -R 50   зміна розмірк панелі на 50 вправо

:new -s new_sesion    Створення сесії з назвою new_sesion 

s менеджер сесій 

Встановлення менеджера плагінів для tmux
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm

.tmux.conf створення файлу конфігурації

Ctrl+b Shift + i  установка плпгінів прописаних в конфігурації

Ctrl+b d вийти без закриття сесій

Ctrl+b $ переіменовати сесію


Новая сессия:
tmux

Создать новую сессию с именем :
tmux new -s myname

Подключиться к уже существующей сессии:
tmux a  # (or at, or attach)

Подключиться к существующей сессии с именем :
tmux a -t myname

Просмотреть запущенные сессии:
tmux ls

завершить сессию:
tmux kill-session -t myname
